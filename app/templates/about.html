What is a Currency Arbitrage?

The average currency arbitrage is very technical and therefore requires previous knowlegde of the system to maneuver and gain a profit. We designed a currency arbitrage which allows the average person to easily maneuver and profit by displaying the different arbitrage opportunities to the user with the click of a button.

The system utilizes eight major currencies such as JMD, GBP, EUR, TTD, CHF, CAD, USD and KYD to accept as source currencies and produce a directed graph displaying the shortest paths available and the total rate for the user.


Trading

We provide online access to a currency arbitrage which allows users to spectate the currency arbitrage opportunities provided with the click of a button.

Currency Data & Analytics

The currency layer api trusted by thousands delivers the standard in foreign exchange data via their exchange rates API which is used by our system. The exchange rates API offers exceptional quality, reliability and accuracy.

Bellman Ford Algorithm

The system utilizes the Bellman Ford algorithm to find the shortest path between currencies. It is used in situations where a source currency is selected and the shortest paths to every other currency in the graph needs to be determined.
